算法-插入排序

2021-03-04 | 算法

原理：  

  1. 把所有元素分为两组，已经排序和未排序的
  2. 找到未排序组中的第一个元素，向已经排序的组中插入
  3. 倒叙便利已经排序的元素，依次和待插入的元素进行比较，直到找到一个元素小于等于待插入元素，那么就把待插入元素放到这个位置，其他的元素向后移动一位。

public class InsertionSort {

    public static void srt(Comparable[] a) {
        for (int i = 1; i < a.length; i++) {
           for(int j=i;j>0;j--){
               //比较索引j处的值和索引j-1的值，如果索引j-1处的值大，则交换数据
               if(greater(a[j-1],a[j])){
                    exch(a,j-1,j);
               }else{
                   break;
               }
           }
        }
    }

    //比较v元素是否大于w元素
    private static boolean greater(Comparable v, Comparable w) {
        return v.compareTo(w) > 0;
    }

    //数组元素i和j交换位置
    private static void exch(Comparable[] a, int i, int j) {
        Comparable temp;
        temp = a[i];
        a[i] = a[j];
        a[j] = temp;
    }

}

```  
分析：  
使用了双层for循环，其中内层循环的循环体是真正排序的代码，主要分析内层循环体的执行次数即可。  
比较次数为：  
(N-1)+(N-2)+(N-3)+...+2+1=((N-1)+1)*(N-1)/2=N^2/2-N/2;  
交换次数为：  
(N-1)+(n-2)+(N-3)+...+2+1=((N-1)+1)*(N-1)/2=N^2/2-N/2;  
总执行次数为：  
(N^2/2-n/2)+(N^2/2-N/2)=N^2-N;  
按照大O推导法则，保留函数中的最高阶项，时间复杂度为O(N^2)
